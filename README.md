## Project Description
This project builds a machine learning model that predicts the **price range of mobile phones** based on their hardware specifications. It is a **multi-class classification** task commonly used for pricing strategy, market segmentation, and recommendation systems.

---

## Dataset Description
https://www.kaggle.com/datasets/chandanashwath/mobile-price-train-data
The dataset is sourced from **Kaggle** and contains various mobile phone features such as:

- Battery power  
- RAM  
- Internal memory  
- Screen resolution  
- Camera specifications  
- Connectivity features (3G, 4G, Dual SIM)  

**Target Variable:** `price_range`  
- 0 = Low  
- 1 = Medium  
- 2 = High  
- 3 = Very High  

You can load the dataset directly from Kaggle or download it manually.

> Note: The dataset is not included in this repository and must be imported from Kaggle before running the notebook.
